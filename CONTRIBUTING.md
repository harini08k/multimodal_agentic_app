# Contributing to Agentic App Manager

Thanks for your interest in contributing! This project demonstrates agentic, multimodal AI interactions in Flutter using Firebase AI Logic and Gemini.  

## How to Contribute

1. **Report Issues:**  
   - Open GitHub issues for bugs or suggestions.

2. **Suggest Features:**  
   - Submit detailed feature requests via issues.

3. **Pull Requests:**  
   - Fork the repo and create a branch for your changes.
   - Make sure your code follows existing style and conventions.
   - Test changes locally before submitting a PR.
   - Submit a pull request with a clear description of your changes.

4. **Feedback on Documentation:**  
   - Suggestions to improve README, setup instructions, or examples are welcome.

---

## Code of Conduct

Be respectful and constructive. By contributing, you agree to uphold this code of conduct.
